# CIV-MEM — sub-library of SELF-LIBRARY

**CIV-MEM** (civilizational memory) is a **governed reference domain** inside **SELF-LIBRARY**. It is **not** part of **SELF-KNOWLEDGE** (`self.md` IX-A).

## What lives here

- **LIB entries** in [`../self-library.md`](../self-library.md) whose `scope` includes tags such as `civilization_memory`, `history`, CMC-facing doors.
- **Hybrid corpus:** [`docs/civilization-memory/`](../../../docs/civilization-memory/), encyclopedia regen — see [civ-mem-encyclopedia-hybrid.md](../../../docs/civ-mem-encyclopedia-hybrid.md).
- **Runtime routing:** after SELF-LIBRARY miss, lookup may query **CMC** — i.e. the CIV-MEM reference path, not identity. See [cmc-routing.md](../../../docs/cmc-routing.md).

## Gate

Additions that change Voice lookup perimeter still pass **RECURSION-GATE** + approval. Prefer **`proposal_class: CIV_MEM_ADD`** or **`CIV_MEM_REVISE`** on candidates when staging civ-mem work. See [identity-fork-protocol.md](../../../docs/identity-fork-protocol.md) §3.5.

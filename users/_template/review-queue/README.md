# Template change-review queue (scaffold)

Pre-activation placeholder. Directories `proposals/`, `decisions/`, and `diffs/` may be empty until the instance adds governed-change artifacts.

Validate with empty dirs:

```bash
python3 scripts/validate-change-review.py users/_template/review-queue --allow-empty
```

See [docs/change-review-validation.md](../../docs/change-review-validation.md).

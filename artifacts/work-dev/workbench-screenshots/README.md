# workbench-screenshots (work-dev)

**Purpose:** Optional **local** or **operator** image captures from Workbench [visual inspection](../../../docs/skill-work/work-dev/workbench/VISUAL-INSPECTION-PROTOCOL.md) (browser screenshots, before/after UI). **Not** Record truth, **not** EVIDENCE.

**Usage:** Save PNGs here when you use the Strategy notebook visualizer pilot or other static demos; point `inspection.screenshots` in a [workbench receipt JSON](../../../docs/skill-work/work-dev/workbench/WORKBENCH-RECEIPT-SPEC.md) to repo-root-relative paths under this tree.

If captures must **not** be committed, add a pattern in `.gitignore` (or ignore `*.png` in this directory locally).

**Related:** [workbench-receipts/](../workbench-receipts/README.md)

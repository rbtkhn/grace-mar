# Patch-intake packets (candidate proposals)

Markdown packets produced here are **WORK-layer candidate proposals**. They are **not** approved merges, **not** canonical Record updates, and **not** gate approvals until staged and reviewed per companion workflow.

- **Adapter:** [`scripts/coding_agent_patch_intake.py`](../../scripts/coding_agent_patch_intake.py) — see [`docs/mcp/coding-agent-patch-intake.md`](../../docs/mcp/coding-agent-patch-intake.md).

Generated `*.md` under this folder may be **gitignored**; `.gitkeep` preserves the directory. Packets may inform **`recursion-gate.md`** staging later; merge authority stays outside this tooling.

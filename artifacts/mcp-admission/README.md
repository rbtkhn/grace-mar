# MCP manifest admission packets

Markdown packets produced here are **WORK-layer governance artifacts** from **`scripts/mcp_manifest_admission.py`**. They classify proposed MCP server manifests against **[`config/mcp-capabilities.yaml`](../../config/mcp-capabilities.yaml)** — **without launching MCP servers**, **without approving credentials**, and **without authorizing live execution**.

See **[`docs/mcp/mcp-manifest-admission.md`](../../docs/mcp/mcp-manifest-admission.md)**.

Default generated `*.md` under this folder may be **gitignored**; `.gitkeep` preserves the directory. Passing admission review remains distinct from enabling integrations.

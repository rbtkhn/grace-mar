# Intent coherence checklist (identity vs intent)

Use when **self-work** (operational intent) and **self-identity** / **self-knowledge** (durable Record) might drift.

1. Does this statement describe **who I am or am becoming** (durable) vs **what I am doing this week** (operational)?
2. If it is durable, is it already covered in **self-identity** or **self-knowledge** (or IX) with gate approval?
3. If it is operational only, does it belong in **self-work.md** instead of a Record file?
4. Would I still endorse this after a month away from the project?
5. If two documents disagree, which one is **ephemeral** (**self-memory** / self-work) and which is **gated Record**?

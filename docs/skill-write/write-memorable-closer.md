# WRITE craft — memorable declarative closer (core argument)

**Scope:** Short **public-facing** copy shaped by [write-operator-preferences.md](write-operator-preferences.md) — **Locals** (VivaBarnes, Duran), **X**, **YouTube comments**, and similar surfaces.

**Relation to sibling rules:** This is the **positive** closer target. It **pairs with** [write-no-rhetorical-question-closer.md](write-no-rhetorical-question-closer.md) (no question as last line) and [write-no-abstract-stacked-closers.md](write-no-abstract-stacked-closers.md) (no abstract stack). See [write-shipping-checklist.md](write-shipping-checklist.md) step **4**.

---

## Rule

End on **one strong, memorable sentence** that **encapsulates the core argument** — what the reader should **carry away**.

- **One** sentence (not a closing **paragraph** of recap unless the surface explicitly needs it).
- **Declarative** — states a **thesis** or **tension** in plain prose; see the **no rhetorical question** doc for why the last line should not be a **question**.
- **Concrete** where possible — who, what fork, what cost — so the line **sticks** instead of dissolving into mood.

---

## Avoid as the final line

- **Rhetorical questions** — [write-no-rhetorical-question-closer.md](write-no-rhetorical-question-closer.md)
- **Abstract stacks** — bill + trust + everyone — [write-no-abstract-stacked-closers.md](write-no-abstract-stacked-closers.md)
- **Soft fade-outs** — “Time will tell,” “Only history will say,” **unless** you deliberately want that register (rare for default operator copy)

---

## When this rule flexes

- **X** — the “memorable sentence” may **be** the whole post; still **one** clear takeaway.
- **Thread** — last tweet in a chain carries the **encapsulation**; earlier tweets **earn** it.

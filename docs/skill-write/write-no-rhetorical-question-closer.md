# WRITE craft — no rhetorical question closer

**Scope:** Short **public-facing** copy shaped by [write-operator-preferences.md](write-operator-preferences.md) — primarily **Locals** (VivaBarnes, Duran), **X**, and **YouTube comments**; any similar surface where **clarity** beats **panel-show** rhythm.

**Relation to [write-operator-preferences.md](write-operator-preferences.md):** Craft slice under **Closers and density**; see [write-shipping-checklist.md](write-shipping-checklist.md) step **4**.

---

## Rule

Do **not** end the piece — usually the **last sentence of the final paragraph** — with a **rhetorical question** (*What breaks first…?*, *Who pays…?* as a hook with no answer in the next line).

That register reads like **cable / newsletter compression**, not **declarative** analysis, and is **atypical** of how serious **journal or history** prose typically **closes** an argument.

---

## Prefer instead

- **Declarative** tension: name the **fork** in plain prose (*Two pressures ran in parallel…*).
- **Qualified uncertainty** without a question: *It was not yet clear whether…* / *The open issue was…*
- **Cut** the closer if the prior sentence already lands.

---

## When this rule does not apply

- **Quoted** dialogue, interview, or **deliberate** Socratic framing you **explicitly** chose (rare for default Locals / X / PH operator copy).

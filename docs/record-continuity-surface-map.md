# Record and Continuity Surface Map

**Status:** Boundary map. This document does not create merge authority and does not replace the gated pipeline.

Grace-Mar uses several adjacent surfaces that can look similar during a working session. Keep them separate by asking what kind of continuity each one provides.

| Surface | Canonical path | Authority | Primary job | Do not use it for |
|---------|----------------|-----------|-------------|-------------------|
| SELF | `users/[id]/self.md` | Durable Record | Identity, SELF-KNOWLEDGE, curiosity, personality | Session notes, draft facts, reference corpora |
| SELF-LIBRARY | `users/[id]/self-library.md` | Durable governed reference | Return-to sources, governed shelves, CIV-MEM and operator-book routing | Short-lived session continuity or identity claims |
| EVIDENCE | `users/[id]/self-archive.md` | Durable Record | Activity log, artifact spine, approved evidence | Scratch notes or ungated conclusions |
| self-memory | `users/[id]/self-memory.md` | Runtime continuity, non-Record | Rotatable short/medium/long session context, process pointers, open loops | Durable facts, identity, reference corpora, evidence |
| self-history | `users/[id]/self-history.md` | Derived, non-Record | Dense dual log from WORK history plus gate-approved companion stream | New facts, pending candidates, merge decisions |

## Routing Rule

If the content says **who the companion is**, stage it for SELF through the gate. If it says **what source to return to**, route it to SELF-LIBRARY through the governed library path. If it says **what this session needs to remember**, keep it in self-memory and rotate it. If it says **what happened and was approved**, it belongs in EVIDENCE. If it summarizes already-governed or WORK history for orientation, it may belong in self-history.

## Karajan Boundary

The long-arc shape is:

- MEMORY points toward durable surfaces; it does not duplicate them.
- SELF-LIBRARY holds reference shelves; it does not become session state.
- EVIDENCE preserves approved activity; it does not absorb runtime residue.
- self-history narrates from existing sources; it does not promote pending material.

When a line could fit more than one surface, prefer the surface with the narrower authority. Escalate to the gate only when the content would change Record truth.

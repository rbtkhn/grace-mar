# "We did X" → stage: keep the gate current

**Purpose:** Make it a **habit** that after real activity you say **"we did X"** so the agent runs signal detection and stages candidates to RECURSION-GATE. The gate then reflects real activity and doesn't go stale.

**Rule (agents.md):** When the companion says **"we [did X]"**, it is a pipeline invocation. Immediately run signal detection and stage candidates. Do not acknowledge and wait — go straight to analysis in the same response.

---

## When to say "we did X"

After any of these, say it in the same session (or next time you're in Cursor/Telegram):

| Trigger | Example |
|--------|--------|
| **Call or meeting** | "We had a call with [who] about [what]." / "We did a prep using the A/B/C template for [decision]." |
| **Reading or consumption** | "We read [book/article]." / "We listened to [podcast]." / "We watched [thing]." |
| **Decision or outcome** | "We decided [X]." / "We shipped [deliverable]." / "We sent [message/memo]." |
| **Work milestone** | "We finished the scenario brief for [client]." / "We used the three minds in prep for [call]." |
| **Record-worthy moment** | Anything that would enrich the companion's profile (knowledge, curiosity, personality) and isn't yet in SELF or the gate. |

You don't have to name every tiny action. Focus on: **calls, readings, decisions, milestones, and anything that feels like "this should be in the Record or at least in the gate."**

---

## What happens

1. You say **"we did X"** (or equivalent: "we had…", "we read…", "we decided…").
2. The agent treats it as a **pipeline invocation**: runs signal detection on the reported activity.
3. Candidates (knowledge, curiosity, personality) are **staged** to `users/grace-mar/recursion-gate.md`.
4. You **approve or reject** in the weekly review (or when you next process the gate). Until then, they sit in the gate — so the gate **reflects real activity** instead of going stale.

---

## Why it matters

- **Gate stays current** — Pending candidates are real, not forgotten. You're not left with an empty gate while life happened off the books.
- **No silent drift** — If you don't say "we did X," the Record doesn't know. The habit closes the loop.
- **Weekly review is easier** — When you open the gate, you see actual recent activity to approve or reject, not a blank or ancient list.

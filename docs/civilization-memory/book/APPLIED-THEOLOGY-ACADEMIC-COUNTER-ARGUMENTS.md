## Academic domains and thesis targets

| Domain | Theses challenged | Type of counter-argument |
|--------|-------------------|---------------------------|
| Just war / defensive force | Condition, seam, “put down the blade” | Some killing justified (e.g. defence of innocents); opposes strict “abandon the footnote.” |
| Political realism (IR) | Condition, coordination hypothesis | Anarchy, security dilemma; norms follow structure; coordination on “condition” or divine return fragile. |
| Religion and violence | Seam, condition, one subject many tongues | Religion as cause of violence; peacebuilding secondary to political/economic factors. |
| Hermeneutics / violent scripture | Scripture as test | Violent texts as command or historical violence, not as “test”; opposes test reading. |
| Political theology | Heaven as build, one subject many tongues | Who defines “peace”; risk of coercive universalism; “one subject” as hegemony. |
| Philosophy of religion | Condition, heaven as build | Problem of evil, divine hiddenness; Pelagianism / works-righteousness. |
| Game theory / coordination | Coordination hypothesis | Divine return as focal point questioned; multiple equilibria, commitment problems. |
| Anthropology / sociology of religion | Seam, “same structure everywhere” | Seam not uniform across traditions; structure more diverse than claimed. |

# The book

**BOOK.md** is the full manuscript: the six core essays in one file, ordered as a **narrative arc** (option 6):

1. **The Call** — The Simple Condition (letter to religious leaders)
2. **The Response** — One Subject, Many Tongues (Islam and Christianity, shared reference)
3. **The Deepening** — Universal Love and Mastery (love, mastery, attention, memory, monad)
4. **The Science** — The Coordination Hypothesis (installation, dehumanisation, contact, game theory)
5. **The Framework** — The Delusion of Separation (recognition theory, six frequencies)
6. **The Use** — AI Ethics from the Condition (rules for AI and shared projects)
7. **The Coda** — Writing the Book and Death (the most interesting activity; no need to fear death or seek eternal life)

Source essays live in `../essays/`. This file is the single-document edition for reading or export.

---

**APPLIED-THEOLOGY.md** is the combined volume of **all** theological materials in the library: the same narrative arc as BOOK.md (Parts 1–7) plus:

8. **Preserving Consciousness** — Biological mortal, symbolic immortal; keep the species writing.
9. **Scripture as Test** — Prophetic scriptures as test; winning by refusing to act out the script; application to Christianity, Islam, Hinduism, Buddhism; tension with prophecy-fulfillment (e.g. Zionism).
10. **Appendix: God is non-zero-sum** — The good is not scarce; be fruitful and multiply; expand the light of consciousness.

Source: essays in `../essays/`, notes in `../notes/`. Single-document edition for reading or export.

**website/** — Static website that presents the book: run `python3 build.py` in `website/` to generate `index.html` from APPLIED-THEOLOGY.md; open `index.html` or serve the folder to view. See `website/README.md`.

**beauty-and-the-blade-manuscript.md** is the publisher-ready manuscript for *Beauty and the Blade* (*A Symphony of Civilizations*): same body content as **APPLIED-THEOLOGY.md** (library mirror) with title page (author/copyright placeholders), word count, in-text chapter cross-references (no file paths), back-matter **Sources and References**, and manuscript particulars for typesetting. Use this file for submission to a publisher.

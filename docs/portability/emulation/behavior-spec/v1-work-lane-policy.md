# Portable Emulation WORK Lane Policy v1

## Core rule

- WORK is local, non-canonical, and runtime-scoped.

## What WORK may contain

WORK **MAY** contain:

- observations
- drafts
- temporary reasoning
- proposed follow-ups
- local planning notes

## What WORK may not do

- WORK **MUST NOT** overwrite Record.
- WORK **MUST NOT** be represented as approved truth.
- WORK **MUST NOT** imply source-repo merge.

## Returning WORK to Grace-Mar

- WORK may be exported back as runtime observations or proposal envelopes.
- Returned WORK **SHOULD** include source bundle metadata so Grace-Mar can understand which exported state the foreign runtime used.

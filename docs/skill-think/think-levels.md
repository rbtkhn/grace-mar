# THINK — maturity levels

**Purpose:** Describe **how far** a capability claim has been evidenced — not grades and not IX personality labels.

## Levels

| `level` | Meaning |
|---------|---------|
| `exposed` | Has encountered material; minimal performance evidence |
| `emerging` | Repeated successful use with support or narrow context |
| `consistent` | Reliable in typical contexts; knows common failure modes |
| `transferable` | Applies in new contexts with light scaffolding |
| `independent` | Sustains use without operator structuring (still not “identity”) |

**Note:** `independent` in THINK does **not** mean the fact belongs in **IX-A** — only that capability evidence is strong. IX promotion follows [think-to-ix-promotion-rules.md](think-to-ix-promotion-rules.md).

# Sovereignty

**Status:** Doctrine and operator synthesis. Subordinate to `AGENTS.md`, the gated pipeline, and the knowledge boundary.

Grace-Mar is a local-first companion-self instance. Its durable memory, governance, and rebuildable substrate live in the repository. Cloud and frontier models may help think, draft, search, or review, but they are visitors to the stack, not owners of the Record.

## Core Rule

Grace-Mar owns its memory and authority locally. Models may process context, but they do not become the source of truth.

That means:

- SELF and EVIDENCE change only through the gated pipeline.
- SELF-LIBRARY is the governed reference layer, not a vendor memory store.
- self-memory is rotatable continuity, not a substitute Record.
- runtime observations, complements, exports, and dashboards may accelerate work, but they remain outside canonical truth until staged and approved.
- derived artifacts are rebuildable operator surfaces, not authority.

## Cloud as Visitor

Frontier models are useful when the task needs unusual reasoning depth, hard synthesis, external lookup, or coding help. Their output should return with receipts, citations, diffs, or staged candidates. It should not silently overwrite local memory.

Local and deterministic tools are preferred when the task is repetitive, private, context-heavy, rebuildable, or mechanical. This keeps the substrate inspectable and reduces dependency on any one provider.

## Design Test

Before adding a new memory, model, runtime, or agent feature, ask:

1. Where does the durable source of truth live?
2. Can the derived or runtime surface be rebuilt or audited?
3. Does this feature preserve companion approval for Record changes?
4. If a cloud model disappears tomorrow, what local substrate remains?

If the answer depends on vendor memory or unreviewed model output, the design is not sovereign enough for Grace-Mar.

## Non-Goals

- This document does not ban cloud models.
- This document does not create new merge authority.
- This document does not replace model routing YAML, runtime complements policy, or the recursion gate.
- This document does not require local model execution before the rest of the stack is ready.

# Daily Xavier journal inbox (accumulator)

**Purpose:** **Append-only** scratch surface for the **current local calendar day** while you track OB1/Cici, coaching notes, or links — **before** they land in the per-day [`inbox/`](inbox/README.md) file the digest ingests.

**Fold at `dream`:** Synthesize rolling content into **`inbox/YYYY-MM-DD.md`** for that date (merge with anything already there; do not duplicate raw dump). Then run or schedule [`xavier_journal_ob1_digest.py`](../../../../scripts/xavier_journal_ob1_digest.py) as you already do for the day file. The rolling file is **not** automatically cleared each dream; clear or trim manually when you want a fresh buffer.

**Length (scratch section only — below the append line):** When the scratch body exceeds **~8000 characters**, **prune from the top** (oldest lines first) until roughly **≤ ~5000 characters** remain.

**Contrast:** [`inbox/YYYY-MM-DD.md`](inbox/README.md) is the **digest input** for that calendar day. This file is a **volatile buffer** — same pattern as [strategy-notebook daily-strategy-inbox](../../work-strategy/strategy-notebook/daily-strategy-inbox.md).

**Git:** Commits preserve history.

# BrewMind — visual style guide (WORK)

> **Purpose:** Single reference for **consistent graphic design** across BrewMind web, social, print (e.g. partner pamphlets), and partner kits. **Not** Record; operator/advisor artifact under [work-xavier](README.md).  
> **Live reference:** [BrewMind Café (GitHub Pages)](https://xavier-x01.github.io/brew_mind/) — when in doubt, match the hero and global chrome. **Optional:** sample exact **hex/RGB** from deployed CSS and add a subsection below.

---

## 1. Brand feel

- **Category:** Specialty **café** × **approachable edtech** — warm enough for a drink brand, structured enough to feel like a **credible learning product**.
- **Keywords:** Grounded, intelligent, **outcome-led**, community, premium-but-accessible.
- **Avoid:** Generic “AI slop” purple gradients, stock-tech clichés, or playful cartoon coffee that reads **childish** next to “learn to earn.”

---

## 2. Color palette

| Role | Description | Usage |
|------|-------------|--------|
| **Primary ground** | Deep **forest green** — matte, rich | Page background, pamphlet base, large fields |
| **Accent** | Warm **gold / amber** | CTAs, emphasized headline words, step numbers, key icons |
| **Primary text on dark** | **White** | Headlines, body on green |
| **Atmosphere** | **Softer green** circles or shapes, low opacity | Background depth only — must not fight text |

**Contrast:** Keep body text **highly readable** on green; use gold for **accents**, not long paragraphs.

---

## 3. Typography

| Use | Style | Notes |
|-----|--------|--------|
| **Display / hero headline** | **Bold high-contrast serif** | Trust + personality; pair with one **italic** phrase in **accent gold** for emphasis |
| **UI, body, navigation** | **Clean geometric sans-serif** | Legibility, “product” clarity |
| **Proof / stats** | Sans, **bold** for numbers | Short labels (“Learning tracks,” “Modules”) |

**Hierarchy:** One strong serif moment per composition; everything else stays **simple sans**.

---

## 4. Layout and UI patterns (digital)

- **Glass panel:** **Frosted / semi-transparent** card on the green ground, **subtle light border** — use for **step-by-step journeys**, feature summaries, or “how it works.”
- **Primary CTA:** **Pill / rounded** button, **solid gold**, soft outer **glow** optional (keep subtle in print).
- **Secondary CTA:** **Ghost button** — transparent fill, **thin white or light** outline.
- **Steps:** **Numbered circles** in accent color; short, scannable lines (buy → learn → earn).
- **Logo mark:** Small **coffee cup** icon acceptable as **secondary** brand anchor next to wordmark.

---

## 5. Print and partner materials

- **Pamphlets / flyers:** Lead with **forest green** + **white** type; use **gold** for **headline accent**, **CTAs**, and **QR frame** if applicable.
- **Glass in print:** Suggest with **semi-opaque white or light-green panel** (e.g. 85–92% opacity) over green — not actual blur unless production budget allows.
- **QR + partner context:** Style QR containers with same **border + corner radius** language as site cards for **instant brand recognition**.

---

## 6. Imagery and iconography

- **Icons:** **Flat**, simple; numbered journey steps > decorative illustration clutter.
- **Photography (if used):** Real café light, **Cebu / community** authenticity over generic stock; align with **opening soon / local** story on site.

---

## 7. Copy alignment (visual + words)

- **Direct verbs** and **clear outcomes** (learn, earn, start free) match the **confident** layout — avoid tiny hesitant type under loud headlines.
- **Stats** (tracks, modules, price anchors) should look **intentional** — same type scale as site stats row.

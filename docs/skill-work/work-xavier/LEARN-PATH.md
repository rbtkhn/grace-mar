# Xavier Learn Path (no-terminal first)

**Audience:** Xavier as a new Cursor user  
**Goal:** Use the companion-xavier system safely and effectively without requiring terminal skills on day one.

---

## Path map (4 stages)

1. **Stage A — Orientation (Day 1, 30-45 min)**
   - Read core purpose + boundary docs
   - Complete Session 0 capture
   - Understand gate flow (no hand merge)

2. **Stage B — Guided operations (Week 1)**
   - Run daily no-terminal checklist
   - Draft with prompt patterns
   - Apply pre-post and stress-test checklists

3. **Stage C — Weekly execution (Week 2-4)**
   - Follow weekly execution plan
   - Maintain KPI + budget log
   - Use weekly retro loop

4. **Stage D — Advanced operator (Month 2+)**
   - Understand lane boundaries, escalation, and mirror governance
   - Optional terminal literacy track (not required for basic success)

## Completion signals

- Stage A complete when Session 0 capture is filled and first candidate set is staged.
- Stage B complete when Xavier can run one full post cycle with checklists and manual ship.
- Stage C complete when one full week closes with KPI + budget + retro entries.
- Stage D complete when Xavier can explain mirror scope, gate flow, and escalation rules unaided.

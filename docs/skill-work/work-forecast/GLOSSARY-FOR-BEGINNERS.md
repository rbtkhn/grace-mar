# Forecast Glossary for Beginners

## Forecast

A forecast is a structured guess about what may happen next in a time-based series.

## Time series

A time series is a list of values observed over time.
Examples:

- messages per day
- notes per week
- policy events per month

## Horizon

The horizon is how far ahead the forecast goes.
Example: A 14-day horizon means the forecast covers the next 14 days.

## Point forecast

A point forecast gives one predicted value for each future step.

## Quantile forecast

A quantile forecast gives a range-like view of uncertainty.
This is useful when a single number would be misleading.

## Covariate

A covariate is an outside factor that may affect the series.
Examples:

- school schedule
- holiday
- publication day
- product launch

## Baseline

A baseline is a simple forecasting method used for comparison.
Examples:

- repeat the last value
- use the recent average

## Benchmark

A benchmark compares methods to see whether a more complex method actually helps.

## Assumption

An assumption is a condition the forecast quietly relies on.
Good practice is to state these explicitly.

## Invalidator

An invalidator is something that would make the forecast less trustworthy.

## Artifact

An artifact is a saved output file produced by work in Grace-Mar.

## Receipt

A receipt is a legibility record showing that a forecast run happened and where its outputs live.

## Record vs WORK

Record surfaces are durable and governed.
WORK surfaces are provisional and used for planning, experimentation, and execution.
Forecasting belongs in WORK unless a human later promotes a conclusion through review.

# Emerging patterns (generic)

**Status:** WORK only — template discipline.  
**Scope:** Lane-level habit for **early, incomplete** patterns that might matter strategically or operationally.  
**Record boundary:** No direct Record, SELF, EVIDENCE, or Voice writes. **Promoted-to-Record** requires RECURSION-GATE + companion approval.

**Reference implementation:** [weak-signals.md](../work-strategy/weak-signals.md) (work-strategy).

---

## Purpose

An **emerging pattern** is a low-to-medium-confidence signal that is not yet strong enough to treat as doctrine, a stable lead theme, or a finished conclusion.

This discipline reduces:

1. **Missing real change early** because the pattern still looks fragmentary.  
2. **Overclaiming from noise** by turning suggestive inputs into false certainty.  

Test meaningful patterns with a **framing audit** ([framing-audit-template.md](framing-audit-template.md)) before they harden.

---

## Definitions

### Emerging pattern

Early evidence for a structure that **might** matter if it persists or strengthens.

### Framing audit

A short test of a proposed frame, analogy, or model (fit, mismatch, falsifier). Not permission to force-fit reality to a template.

### Promotion (to ledger)

Move a surviving pattern into **WORK-LEDGER** §II-A or equivalent when it recurs, strengthens, or changes priorities. See [README glossary](README.md#glossary-lifecycle) — **promoted-to-ledger** vs **promoted-to-Record**.

### Retirement

Retire when the falsifier fires, recurrence stops, explanatory power drops, or a broader pattern subsumes it.

---

## Confidence

Use only **low** or **medium** while the item is still an “emerging pattern.” If confidence becomes **high**, it is probably time to promote to ledger, fold into a lead theme, or drop the “emerging” label.

---

## Status values (suggested)

- **watch**  
- **escalating**  
- **promoted-to-ledger** (recorded in WORK-LEDGER or lane equivalent)  
- **retired**  

Lanes may align wording with existing habits (e.g. work-strategy uses **promoted** toward STRATEGY.md — same idea as promoted-to-ledger).

---

## Daily / cadence rule (optional)

Some lanes embed **one** emerging-pattern block in a **daily brief** ([daily-brief-template.md](daily-brief-template.md)). If nothing clears the bar, say so explicitly rather than filler.

---

## Operator standard

Notice early → compare carefully → preserve contradictions → promote slowly → retire honestly.

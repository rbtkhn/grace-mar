# TERMS — Xavier instance + work-cici (short)

| Term | Meaning |
|------|---------|
| **Companion** | The human whose Record this is (Xavier). |
| **Record** | Documented self — `self.md`, evidence, library, skills — **gated** merges only. |
| **Voice** | Queryable emulation of the Record (bot) — reactive; does not replace the Record. |
| **WORK** | Job/client/campaign lanes — **not** pasted into SELF without pipeline. |
| **SELF-LIBRARY** | Reference-facing shelf — **not** SELF-KNOWLEDGE (IX). |
| **RECURSION-GATE** | Where candidates wait for **approve** before merge. |

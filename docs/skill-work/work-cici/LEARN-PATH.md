## Path map (4 stages)

1. **Stage A — Orientation (Day 1, 30-45 min)**
   - Read core purpose + boundary docs
   - Complete Session 0 capture
   - Initialize `work-business/xavier` starter pack from survey + business docs
   - Understand gate flow (no hand merge)

2. **Stage B — Guided operations (Week 1)**
   - Run daily no-terminal checklist
   - Draft with prompt patterns
   - Apply pre-post and stress-test checklists

3. **Stage C — Weekly execution (Week 2-4)**
   - Follow weekly execution plan
   - Maintain KPI + budget log
   - Use weekly retro loop

4. **Stage D — Advanced operator (Month 2+)**
   - Understand lane boundaries, escalation, and mirror governance
   - Optional terminal literacy track (not required for basic success)

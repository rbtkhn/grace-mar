4. Non-negotiables (legal / ethical posture)

Do Don’t
Post only reviewed content you stand behind. Autonomous posting by bots or scheduled dumps without your pass.
Tie factual claims to sources (news, Congress.gov, Massie’s own posts). Invent positions Massie didn’t take.
Label analysis as unofficial when context needs it. Speak as Massie, his campaign manager, or his office.
Escalate if a draft feels legally or reputationally hot. Promise votes, endorsements, or deals on his behalf.

FEC / campaign law: Handled by any official campaign counsel; this account is not that. If unsure, don’t post — ask the companion.

The operator may run an internal daily brief script and a Massie news scan (draft post ideas). You don’t run those scripts unless the companion asks you to use Cursor — but you consume the outputs:

Input What you do
Daily brief file (if shared) Scan for priorities, risks, and what Massie did/said.
Draft post candidates (from agent) Run through smm-training checklist; edit; post or reject.
Internal WAP dashboard (/wap on miniapp host, token from operator) Optional: log jobs and paste Cursor outputs so work isn’t lost when a chat thread closes. See wap-dashboard.md.

# Social Media Playbook — Simple Condition Public Awareness ($1K)

**Purpose:** One-page guide for organic posting and paid boost under the [budget plan](budget-plan-1000.md). Use with [employee-instructions-1000-budget.md](employee-instructions-1000-budget.md) Step 1.4 (organic) and Step 2.1 (paid). **Principal approves channels and posts before publish.**

---

## Recommended platforms (for $1K plan)

| Priority | Platform | Use |
|----------|----------|-----|
| **1** | **Meta (Facebook and/or Instagram)** | Primary for organic + paid. Quote graphics and short video work on both. Use same post to boost in Phase 2. |
| **2** | **X (Twitter)** | Quote graphics + link; good for secular/coordination and faith-adjacent audiences. Optional paid promote ($50–75). |
| **3** | **Threads** | Cross-post from Instagram if org has account; same asset, no extra production. |
| **Optional** | LinkedIn | If audience includes educators, clergy, or policy: one post with one-pager or letter-summary link. More formal tone. |
| **Optional** | TikTok | Only if video is ready and account exists; 60–90 sec script fits. Not required for $1K. |

**Which accounts:** Principal decides — org account, principal’s personal, or both. Use the same 3–5 assets across chosen platforms so “best performer” is comparable.

---

## Posting cadence (Phase 1 — Seed)

- **Total:** 3–5 posts over 2–3 weeks (before paid boost).
- **Pace:** 1–2 per week so you have time to see which post gets the most saves, shares, or comments.
- **Spread:** Use a different quote graphic per post (or repeat one top quote once). Vary the caption (see below).
- **CTA:** Every post ends with a clear ask: “Read the full letter” + [link to letter or one-pager]. Add link in bio or first comment if platform limits link in post.

## Replies and DMs

- **Replies:** Reply to thoughtful comments (thanks, short clarification, or “Read the letter and see what you think”). Don’t argue; point to the letter.
- **DMs:** If someone DMs with interest, send the link to the letter or one-pager and one sentence: “Would love to hear what you think after you read it.” No mass DMs from this playbook; clergy outreach is separate (Step 1.5).

---

## Paid boost (Phase 2 — reminder)

- **Which post:** Use the organic post that got the most engagement (saves, shares, comments, or link clicks). If none stood out, principal picks one.
- **Platform:** Meta first ($75–100, 7–10 days); optionally X ($50–75, 2–3 days). Same creative + link.
- **Targeting (Meta):** Interests such as religion, peace, interfaith, philosophy, theology. Geography per principal.
- **Approval:** Principal approves platform, post, targeting, and spend before the boost runs. Log results in [results-log.md](1000-budget-assets/results-log.md).

# 48-Hour Virality Sprint

Compressed execution for maximum reach within 48 hours. No waiting for Congress vote or anniversary — **the drop itself is the focal event.** Human approval required before any public ship.

---

## Core principle

In 48 hours you cannot build new audiences. You **activate existing networks** with one clear ask and one shareable payload. Focal point = "Demand the vote. End the Iran war. Share this."

---

## Hour 0–12: Ship the payload

| Task | Output | Owner |
|------|--------|-------|
| **Lock 5 one-liners** | Pick from [one-line-library](one-line-library.md): 1 peace-as-condition, 1 see-the-face, 1 demand-the-vote, 1 blade/beauty, 1 waiting | Operator |
| **Create 2–3 graphics** | Quote + minimal design (canva/figma). No custom imagery needed — text-only works. | Designer / operator |
| **1 "see the face" asset** | Single photo + caption OR 30-sec video. Source: verified, reputable. Iranian person/family — name, one sentence. | Operator |
| **Finalise CTA** | "Contact your rep" link (e.g. 5calls.org, Congress lookup) + #DemandTheVote #EndTheWar | Operator |

**Do not over-produce.** Text-only graphics, one image, one link. Ship by hour 12.

---

## Hour 12–24: Activate networks

| Task | Action | Target |
|------|--------|--------|
| **DM 10–20 influencers** | Send [influencer-brief](influencer-brief.md) + 2–3 ready-to-post graphics. "Post in next 24–48 hrs. No commitment beyond one post." | Clergy, peace creators, bipartisan, cultural |
| **Seed 3–5 posts** | Operator or trusted account posts; tag 2–3 people who might amplify | Personal network |
| **UGC prompt** | "Reply with one name of someone waiting. We'll add it to the reel." — sticky, low-friction participation | Any platform |
| **Cross-post** | Same content: X/Twitter, Instagram, Threads, TikTok (if format fits), LinkedIn | All owned channels |

**Prioritise DMs over cold posting.** Personal ask beats algorithmic hope.

---

## Hour 24–48: Amplify and measure

| Task | Action |
|------|--------|
| **Repost / quote** | Engage every reply, quote-tweet or repost strong takes, thank amplifiers |
| **Collate UGC** | "Name one person waiting" — aggregate replies into a "we see them" thread or story |
| **Track** | Impressions, shares, replies, link-clicks, new followers. Note which one-liner and format performed best |
| **Second wave** | If one format wins, create 2–3 variations and re-seed to non-responders |

---

## What to cut (48-hour reality check)

- **No A/B testing** — pick one line per slot, ship it
- **No clergy sign-on letter** — too slow; use "post this graphic" ask instead
- **No custom video** — unless you already have it; 30-sec max
- **No new website** — use existing tools (5calls, Congress lookup, petition link if you have one)
- **No paid boost yet** — organic first; paid only if organic proof of concept

## Success metrics (48-hour)

- **Reach:** Combined impressions across posts (target: 10K+ if you have any network; 1K+ if starting cold)
- **Engagement:** Shares > likes (viral = propagation, not consumption)
- **Link clicks:** Rep contact / petition — action matters
- **UGC:** At least 5–10 "name one person waiting" replies — proves participation, not just views

## Framing / perspective

**The backstory for the gig:** A human received a phone call and was told to post this exact ad. She posted it. That's the perspective—origin obscured. The gig listing (title, description, deliverables below) *is* that ad. When posting on Fiverr, the companion is that human: she was instructed, she posted. No further origin in the narrative. The buyer sees an ad; they don't see Cursor, internal tooling, or system labels. They see a person who posted what she was told to post.

## Detailed instructions and specifications

**For the companion (posting the gig):**
1. Log in to Fiverr. Create a new gig (or duplicate and edit).
2. Paste the **Gig title** into the title field.
3. Paste the **Meta description** into the search/summary field (if Fiverr has one; else use the first 120 chars of the full description).
4. Paste the **Full description** into the main description area. Fix any line breaks if Fiverr strips them.
5. Set **Category:** Writing & Translation → Research & Summaries (or "Creative Writing" or closest match).
6. Set **Delivery time:** 2 days.
7. Set **Price:** $100 for the basic package.
8. Add **Extras** if Fiverr allows: "+$50 — 5 draft tweets"; "+$50 — Second opponent or issue."
9. In **Requirements**, add the four questions above (candidate, opponent, race/context, optional links). Mark first three required.
10. Upload 1–3 images if needed (e.g. a simple "Campaign one-pager — $100" graphic, or leave default).
11. Publish. Save the gig URL in this doc or in revenue-log.

**When an order comes in:**
1. Open the order. Read the buyer's answers (candidate, opponent, race/context, links).
2. In Cursor (or your workflow), say: "Fiverr order: [paste the buyer's answers]. Produce the $100 one-pager."
3. Agent produces the one-pager (research, structure, message angles, citations). Use public sources only; cite everything.
4. Companion reviews for tone, accuracy, and that it matches the buyer's context. Edit if needed.
5. Export to PDF (or create a Google Doc and set to "Anyone with link can view"; copy link).
6. In Fiverr, go to the order → Deliver order → Attach the PDF (or paste the Doc link) and a short message: "Here's your one-pager. One revision included if you'd like any changes."
7. Mark as delivered. If buyer requests revision, repeat steps 2–6 with their notes and re-deliver.
8. Optionally log in revenue-log: date, $100, type "Fiverr microtask — campaign one-pager."

**One-pager format (spec for the agent when producing):**
- One page (or up to 1.5 if needed for readability). Single-spaced, clear headings.
- **Section 1 — [Candidate/Cause name]:** 2–4 sentences. Key positions, strengths, public record. Cite sources.
- **Section 2 — [Opponent/Issue]:** 2–4 sentences. Summary, positions, vulnerabilities or angles. Cite sources.
- **Section 3 — Three message angles:** For each angle: title (e.g. "Accountability" or "Experience"); 1–2 talking points (1–2 sentences each). Brief, usable.
- **Sources:** Footnotes or a short "Sources" list at bottom (links or citations).
- No fluff; no unsourced claims. Buyer can hand this to staff or use it in the next 24 hours.

# $1,000 Budget Campaign — In-Cursor Assets

**Purpose:** Copy, briefs, templates, and logs for the Simple Condition public awareness campaign. Created in-repo; no spend. See [employee-instructions-1000-budget.md](../employee-instructions-1000-budget.md) and [budget-plan-1000.md](../budget-plan-1000.md).

| File | Use |
|------|-----|
| `one-pager-copy.md` | Copy for designer → one-page PDF. Get principal approval before sending to freelancer. |
| `letter-summary-copy.md` | Copy for designer → 2–3 page letter summary PDF. Get principal approval before sending. |
| `designer-brief-simple-condition.md` | Brief for freelancer: deliverables, quote lines, style, file names. Attach copy files when hiring. |
| `video-script-90sec.md` | Script and production notes for 60–90 sec video. Optional; use in Canva/CapCut or with low-cost editor. |
| `mail-list-template.csv` | Add 50–75 religious leaders/institutions. Columns: Name, Title, Institution, Address, City, State, ZIP, Email, Tradition_Notes, DateSent, Notes. Starter rows included. |
| `spend-log.csv` | Log every spend (date, category, line item, amount, approved by). Total must not exceed $1,000 without approval. |
| `results-log.md` | Log mail sends, email stats, organic social, paid results, clergy outreach. Update as campaign runs. |

**One-line library (parent folder):** [one-line-library-awareness.md](../one-line-library-awareness.md) — 20 shareable lines for awareness campaign (no Iran CTA).

# SMM Day 1 checklist

**Purpose:** Clear runbook for your first day. Check off as you go.

---

## Before you start

- [ ] You have X (Twitter) account access for @shadowcampain (login, 2FA if applicable)
- [ ] You have access to this repo or the onboarding docs (companion provides)

---

## Morning: orientation

- [ ] Read [smm-onboarding-packet.md](smm-onboarding-packet.md) top to bottom
- [ ] Read [account-x.md](account-x.md) — account spec, guidelines, X strategy
- [ ] Skim [principal-profile.md](principal-profile.md) — who Massie is, key issues
- [ ] Skim [smm-training.md](smm-training.md) — voice section and review checklist

---

## Midday: access and baseline

- [ ] Log into @shadowcampain on X; confirm you can view and post
- [ ] Capture baseline metrics (companion or you): follower count, recent engagement — for "before/after" in 30 days

---

## Afternoon: first posts (optional)

- [ ] Review 3 pre-queued draft posts (companion provides; or request from agent)
- [ ] Run each draft through the [smm-training checklist](smm-training.md#checklist-for-smm-when-reviewing-drafts)
- [ ] Post 1–2 approved posts (or schedule if preferred)
- [ ] Optional: Try one calibration post (e.g. narrative-action alignment) — see smm-training calibration exercise

---

## End of day

- [ ] Confirm workflow: how do you request new drafts? (Companion, Telegram, etc.)
- [ ] Note any questions for companion
- [ ] You're set for Week 1

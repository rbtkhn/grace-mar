# work-strategy — notebook lane

**Parent:** [Work notebook (dev-notebook/)](../README.md)

**Holds** optional **paste-ready prompts**, checklists, or spec fragments for strategy work that should stay **separate** from the rolling [strategy-notebook/](../../../work-strategy/strategy-notebook/) (inbox, `days.md`, `raw-input/`, clusters). Add `*.md` here when a capture is a **reusable template**, not a calendar-bound day file.

**Empty by default** — append files when the operator or agent needs a stable hook without touching the main notebook tree.

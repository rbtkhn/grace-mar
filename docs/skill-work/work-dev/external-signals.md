## Guardrail

**Temperature, not requirements.** If Jensen or a pod says “everyone will run X,” your **ship list** still comes from **implemented paths** in-repo and documented **gaps**. Use external signals to **name** buyer anxiety and **align** vocabulary — not to slip undocumented features into the integration story.

# Message Assist Calibration

**Purpose:** Capture operator feedback when message-assist produces a poor draft. Encodes tribal knowledge so future runs improve. Lazar: "How can I prompt you better?" → put learnings into rules.

**Usage:** When a draft is off (wrong tone, missing fact, too formal, etc.), add a calibration note below. The message-assist script loads this file if present and includes it in the prompt context.

**Maintenance:** Companion approves. Add notes only when companion documents what went wrong. Remove or update when a note is no longer relevant.

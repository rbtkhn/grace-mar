# Polyphonic Cognition Lattice
<!-- word_count: 140 -->

WORK only; not Record.

**Doctrine:** a polyphonic cognition lattice is the notebook pattern that emerges when multiple expert lanes keep their own voice, but recurring markers line up across them.

## Use

- Keep each lane distinct.
- Look for shared constraints, not shared phrasing.
- Treat recurrence as a clue to structure, not as automatic agreement.
- Preserve differences in tone, authority, and scope.

## What it is not

- Not a forced synthesis.
- Not a collapse of one voice into another.
- Not a claim that separate analysts are making the same argument.

## Practical test

If Pape, Mercouris, and Alkorshid all point to the same material pressure, but each from a different lane of evidence, the lattice is active. The notebook should surface that overlap while still keeping the sources separate.

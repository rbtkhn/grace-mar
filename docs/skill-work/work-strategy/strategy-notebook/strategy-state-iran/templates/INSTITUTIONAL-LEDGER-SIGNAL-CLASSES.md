# Institutional ledger — signal-class headings (template)

**Role:** In [`weave/`*`-institutional-ledger.md`](../weave/) and new channel month notes, **default** the top organization to **signal class**, not to **`### Voice — …`** (person-primary).

**Default signal classes (use as `### Signal class — <label>`):**

- **negotiation posture** — terms, “ball in X court,” talk/no-talk, Islamabad / JCPOA-adjacent frames
- **deterrence / escalation** — red lines, “cards on the battlefield,” strike vocabulary
- **legitimacy / mourning** — religious-national dates, ROME/UNGA legitimacy wedges, Minab-style grief frames
- **commercial passage / Hormuz** — strait, PMO route, “open for commercial” vs **material** AIS/shipping
- **foreign alignment** — multipolar shout-outs, delegation visits, **Pakistan** / **Munir** lines
- **domestic mobilization** — mass hashtags, council rhetoric, Majlis **coercion** vocabulary

**Other:** `### Signal class — other: <short label>` when nothing above fits; keep the label **grep-friendly** (kebab or plain words).

**Inside each section (required order):**

1. **Emitting office(s):** (e.g. FM account, Presidency, Majlis speaker, repost via `@…`)
2. **Source tier:** (e.g. primary English MFA post, fa-only + machine gloss, Tasnim echo, repost chain depth N)
3. **Bullets** of what was said / linked (pointer to `raw-input` row or bundle)
4. Optional: paste [ANNOUNCED-VS-MATERIAL-BLOCK](ANNOUNCED-VS-MATERIAL-BLOCK.md) when **posture** and **field state** can diverge

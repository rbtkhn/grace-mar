**How this differs from `experts/`:** Commentator threads = **analytic** voice + `thread:`. **Channels** = **which institution** a signal **routes through** and what it is **strong/weak** evidence for (see each `profile.md`).

**Optional month notes:** Add `channels/<name>/YYYY-MM.md` if you want a **short** per-channel month stub; otherwise the month can live in [`weave/YYYY-MM-institutional-ledger.md`](../weave/2026-04-institutional-ledger.md) only. **If five files is too many**, one combined `channels/YYYY-MM-snapshot.md` is acceptable (document the choice in the file header).

# work-strategy — promotion ladder

**Status:** WORK only. **Not** Record truth.

**Authority boundary:** This ladder governs **operator strategy memory** ([STRATEGY.md](STRATEGY.md), strategy-notebook, decision points). It does **not** grant merge authority into **SELF**, **EVIDENCE**, or **prompt**. Anything entering the **Record** still passes **RECURSION-GATE** and companion approval ([AGENTS.md](../../../AGENTS.md)).

## Federation

- **Daily judgment:** [strategy-notebook/](strategy-notebook/README.md), [STRATEGY-NOTEBOOK-ARCHITECTURE.md](strategy-notebook/STRATEGY-NOTEBOOK-ARCHITECTURE.md) (promotion cadence, condense).
- **Synthesis L4 (promotion intent):** [SYNTHESIS-OPERATING-MODEL.md](strategy-notebook/SYNTHESIS-OPERATING-MODEL.md).
- **Watches:** [STRATEGY.md §II-A](STRATEGY.md) active watches.
- **Escalation to structured choice:** [decision-point-template.md](decision-point-template.md), [decision-points/](decision-points/README.md).

## Stages (ordered)

| Stage | Description | Typical location |
|-------|-------------|------------------|
| **0. Inbox lines** | Volatile **paste-ready** scratch (X / YT / links) before synthesis. | [daily-strategy-inbox.md](strategy-notebook/daily-strategy-inbox.md) (format § *Paste-ready one-liner*) |
| **1. Notebook note** | Dated observation in `days.md` — Signal / Judgment / Open — usually produced by folding inbox at **`strategy`** / **`dream`**. | `strategy-notebook/chapters/YYYY-MM/days.md` |
| **2. Active watch** | Tracked hypothesis or risk in STRATEGY §II-A or weak-signal path. | [STRATEGY.md](STRATEGY.md), [weak-signals.md](weak-signals.md) |
| **3. Decision point** | Watch escalates; structured options before commitment. | [decision-points/](decision-points/) instance files |
| **4. Recommendation** | Chosen option + falsifiers; still WORK. | Same decision file + notebook note |
| **5. Promoted watch / cross-lane commitment** | Durable line in STRATEGY §II-A / §IV or explicit cross-lane handoff doc. | STRATEGY.md, lane README |
| **6. Archived / resolved** | Watch closed; decision marked resolved; optional pointer in Open. | STRATEGY.md, decision file `Status` |

Promotion **up** the ladder is intentional and logged; demotion or retirement follows [weak-signals.md](weak-signals.md) and [watch-promotion-rules.md](watch-promotion-rules.md). Moving from **0 → 1** is the normal **inbox fold** (not a separate “promotion” ceremony unless you treat it that way in session notes).

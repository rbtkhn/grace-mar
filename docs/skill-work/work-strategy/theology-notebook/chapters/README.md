# Chapters (optional)

Use this folder for **syntheses** (monthly or thematic) if you outgrow the research/ + ideas/ split. Not required to start the book.

**Possible shapes**

- `YYYY-MM-meta.md` — a short “what moved this month” for your theology work.
- `synthesis-YYYY.md` — year-end or milestone essay.

If unused, leave this README as a placeholder.

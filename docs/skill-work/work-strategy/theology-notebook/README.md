# Theology notebook

## Mission

**This book exists to define your own theological beliefs through the work of writing it.** Research, reading, and rough ideas are not a prelude to a separate “real” catechism somewhere else — the **activity of creating** these files (sorting, arguing with yourself, citing, revising) is how the position takes shape. What lands in the **Record** later still has to pass the gate; this tree is where belief becomes **legible to you** first.

**Book (practically):** A workspace for **theology** — research notes, reading traces, and **ideas** that sharpen or complicate that self-definition, whether or not they are ever promoted to another book or the Record.

## Conventions (lightweight)

- **Gated Record:** Durable **identity** or **IX** claims go through the gate; this tree is a **sketch pad**.
- **Citations:** When a note becomes load-bearing, add enough **source** for a future `verify:` or `Links` line in strategy; avoid unsourced assertive claims in public-facing copy.
- **Naming:** Use clear filenames (`2026-04-26-trinity-note.md` or `topic-slug.md`); no required schema for v0.

## Etsy & shop operations

| Item | When | Action |
|------|------|--------|
| **Star Seller / account health** | Monthly (Etsy’s cycle) | Check message response rate, on-time shipping, review score. Adjust so you don’t lose Star Seller. |
| **Listing refresh** | Quarterly or twice yearly | Review titles, tags, photos, and pricing on key listings; update as needed. |
| **Vacation mode** | Before/after travel or busy periods | Turn vacation mode on/off; set announcement if desired. |
| **Policy review** | Yearly | Review shipping, returns, customs, processing times so they match how you actually operate. |

# Work lane contract (grace-mar)

**Purpose:** Minimum **declared** shape for each `work-*` territory so WORK stays **inspectable** without pretending to be Record.

## Required elements (per lane)

1. **Purpose** — README first section.
2. **Boundary** — explicit “not Record” / gate rules pointer.
3. **Authorized sources** — hand-maintained list or lane-specific `*-sources.md` (see [work-modules-sources-principle.md](work-modules-sources-principle.md)).
4. **Append-only history** — e.g. `work-*-history.md` or territory equivalent.
5. **Open gaps / issues** — known limitations (table or doc).
6. **Promotion / escalation** — what “ship” means and when to touch **RECURSION-GATE**.

**Authority:** Lanes **stage**; companion **merges** Record facts. See [AGENTS.md](../../AGENTS.md).

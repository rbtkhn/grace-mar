# Newsletter digests

Weekly digests are generated by `.github/workflows/newsletter.yml` (cron: Monday 09:00 UTC) and committed here.

- `digest-YYYY-MM-DD.md` — Markdown digest for that week
- `digest-latest.html` — Latest HTML digest (overwritten each run)

To generate locally: `python scripts/generate_newsletter.py --output newsletter/digest-sample.md`

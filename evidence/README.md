# `evidence/` (repository root)

This directory is a **placeholder** for the [state model](../docs/state-model.md) **evidence layer** at the grace-mar **repo root**. It is **not** the default location for operator uploads, screenshots, or WORK ingest — those usually live under territory paths (for example [`docs/skill-work/work-cici/evidence/`](../docs/skill-work/work-cici/evidence/)).

**Why it exists:** Keep the top-level folder tracked (`.gitkeep`) so the name remains available for future routing or parity experiments without implying Cici-style ingestion is wired here.

See [State model — Repo layout (grace-mar)](../docs/state-model.md#repo-layout-grace-mar) and [OB1 bridge mapping — Conceptual map](../docs/integrations/ob1/mapping.md#conceptual-map-ob1-cici-grace-mar).
